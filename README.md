# 测试用例生成辅助工具

这是一个基于AI的测试用例生成辅助工具，可以根据需求描述自动生成测试用例。支持Dify。

## 功能特点

- 支持Dify
- 可自定义测试用例类型比例
- 支持设置测试优先级
- 支持指定生成测试用例数量
- 可导出为Markdown和Excel格式

## 安装步骤

### 1. 安装Python依赖

```bash
pip install -r requirements.txt
```

### 2. 配置模型

#### 配置
- 在Dify创建用例生成的流程，然后在这里配置


### 3. 启动应用

```bash
python run.py
```

## 使用方法

1. 配置选定的模型参数
2. 输入需求描述或上传需求文件
3. 设置高级选项（可选）
   - 调整用例分类占比
   - 设置测试优先级
   - 指定生成测试用例数量
4. 点击"生成测试用例"按钮
5. 查看生成的测试用例
6. 下载测试用例（Markdown或Excel格式）

## 注意事项

- 首次运行会自动创建默认配置文件
- 生成的测试用例仅供参考，可能需要人工调整
- 本地模型的质量和速度取决于硬件配置

## 系统要求

- Python 3.8+
- Windows/Linux/macOS
